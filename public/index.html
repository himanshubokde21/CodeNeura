How it works

From folder to insights in 4 steps

No account. No upload to any server. Your code stays on your machine.

1

Import your project

Click Import and select any local folder. The entire tree is read client-side — nothing is uploaded.

2

Explore the file tree

Navigate your project with a VS Code-style explorer. Click any file to view it with full syntax highlighting.

3

Generate flowcharts & metrics

Instantly see logic flowcharts and deep code metrics — LOC, complexity, quality grade — without any API call.

4

Get AI insights

Hit "Get AI Insights" to let Gemini analyse your code and surface patterns, risks, and improvements.
