About CodeNeura

CodeNeura is a professional-grade AI code visualization tool. Import any project folder to explore its structure, view source files with syntax highlighting, generate instant flowcharts, and get deep code metrics — all powered by Gemini AI.

Features

  • 🌳
    VS Code-style File Explorer
    Color-coded file types, collapsible folders, file count badges, and project stats.
  • ✨
    Syntax Highlighting
    Full Prism.js support for 24+ languages with one-click copy to clipboard.
  • 📊
    Instant Flowcharts
    Client-side Mermaid.js diagrams generated in milliseconds — zero API calls.
  • 📈
    Real Code Metrics
    LOC breakdown, cyclomatic complexity, quality grade (A–F), functions & classes.
  • 🤖
    Gemini AI Insights
    On-demand analysis: summary, design patterns, and actionable improvement suggestions.
  • ⌨️
    Keyboard-First UX
    ⌘K search, resizable panels, SVG export — built for developers.
